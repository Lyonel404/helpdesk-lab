# IT Helpdesk Lab – Portfolio Project

## 📌 Overview
This project simulates real-world IT helpdesk work including troubleshooting, ticket management, and system administration tasks.

---

## 🛠️ Tools Used
- Windows 10/11
- VirtualBox
- Microsoft 365 (simulated)
- GitHub for documentation

---

## 🎯 Skills Demonstrated
- Incident management
- User support troubleshooting
- Network troubleshooting basics
- Documentation of IT issues
- Problem-solving workflow

---

## 📂 Projects
- Ticket system (see /tickets folder)

---

## 🚀 Goal
To demonstrate job-ready helpdesk skills for entry-level IT support roles.
